# Video Transition Files

The homepage already works without any local video files.

If you want real cinematic transitions instead of the built-in animated fallback, drop these files into this folder:

- `01-02-forward.mp4`
- `02-01-reverse.mp4`
- `02-03-forward.mp4`
- `03-02-reverse.mp4`

Expected mapping:

- `01 -> 02`: public speaking to AI
- `02 -> 01`: AI back to public speaking
- `02 -> 03`: AI to music
- `03 -> 02`: music back to AI

Recommended export:

- H.264 MP4
- 1920x1080
- 24fps
- 4-6 Mbps
- short transitions, ideally 2-4 seconds
